**Breed Recognition System** 
  Overview

The Breed Recognition System is an AI-powered web application that identifies cattle and buffalo breeds from images using Convolutional Neural Networks (CNN) and Transfer Learning.
The system not only predicts the breed but also provides detailed breed-specific insights and introduces a Digital Identity system for secure livestock management and trading.

**Key Features**

**AI-Based Breed Recognition**
Image-based classification using CNN + Transfer Learning
Trained on Indian Bovine Breeds Dataset
Fast and accurate predictions

**Detailed Breed Insights**
 Average Milk Production
 Common Diseases
 Origin of Breed
 Average Lifespan
 Height & Weight Range
 Price Range
 Nutrition Requirements
 Dung Production per Day

**Digital Cattle Card**
Generated after breed verification
Contains a Unique Digital ID
Stores all animal-related data securely in database
Enables verification & re-verification by buyers


**Unique Selling Point: Marketplace Module**
🛒 Secure Livestock Marketplace
Only verified cattle with Digital IDs can be listed
Connects authorized sellers & verified buyers
Prevents fraud and fake listings
Maintains ownership history (traceability)

**Usage**
Upload cattle/buffalo image
System predicts breed
View detailed breed information
Generate Digital Cattle Card
